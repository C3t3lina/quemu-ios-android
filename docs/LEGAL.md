## Imágenes de sistema

**Este proyecto NO incluye ninguna imagen de iOS, macOS ni ningún software de Apple.**

El usuario es el único responsable de obtener imágenes de sistema de forma legal:

### Opciones legales:

1. **iOS Simulator (requiere cuenta Apple Developer)**
   - Las imágenes del simulador de Xcode están incluidas en Xcode
   - Su uso personal en un emulador para desarrollo es generalmente aceptable
   - Distribución de dichas imágenes está prohibida por los términos de Apple

2. **Darwin (base open source de macOS/iOS)**
   - Apple publica partes del código de Darwin como open source
   - https://opensource.apple.com/
   - Las imágenes completas de Darwin no están disponibles oficialmente

3. **Alternativas sin Apple:**
   - Ubuntu Server ARM64: https://ubuntu.com/download/server/arm
   - Debian ARM64: https://www.debian.org/ports/arm/
   - Alpine Linux ARM: https://alpinelinux.org/downloads/
